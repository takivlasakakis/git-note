### String manipulation
#### sub title
to manipulate strings as arrays it's best to convert to an array then manipulate and then join them back together again.


```ruby
letters = "hello world!"
letters.split('')
letters.chars
```
=> 
```
["h", "e", "l", "l", "o", " ", "w", "o", "r", "l", "d", "!"]
["h", "e", "l", "l", "o", " ", "w", "o", "r", "l", "d", "!"]
```
